# codespaces-rust

GitHub Codespaces RUST simple guide

# Install Rust

```
wget https://sh.rustup.rs -O rustup.sh
```

```
chmod +x rustup.sh
```

```
./rustup.sh
```

```
source "$HOME/.cargo/env"
```

# Update Rust

```
rustup update
```

# Run Cargo

```
cargo new hello
```

```
cd hello
```

```
cargo run
```
